# AnimatedWidget  :zap:
Flutter AnimatedWidget


## App
Providing simple amination effects to the flutter app using the AnimatedWidget.

## Get Started
Add the following dependencies in pub
```
animated_widgets: ^1.0.6
random_color: ^1.0.5
```
